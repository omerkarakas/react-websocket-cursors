# Instructions to run

1. Clone the project

2. Install the server dependencies

```shell
cd /server
npm i
```

3. Run the server

```shell
 cd /server
 node index.js
```

4. Install the server dependencies

```shell
cd /client
npm i
```

5. Run the client

```shell
cd /client
npm run dev
```

6. Open more than one browsers or browser tabs using different user names
